# Note de partage LinkedIn

Ce dashboard est conçu pour être présenté comme un side project quantitatif.

Positionnement conseillé :
- projet local reproductible de simulation
- couche Monte Carlo pairwise
- couche de propagation complète du tournoi
- interface analytique bilingue
- vue des écarts modèle-vs-marché
- limites explicites et refus de toute fausse certitude

Ce que ce n’est pas :
- un outil d’automatisation de paris
- un moteur de garantie
- un substitut aux données officielles finales ou aux listes définitives
