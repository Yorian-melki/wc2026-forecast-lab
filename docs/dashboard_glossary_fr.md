# Glossaire du dashboard WC2026

Ce glossaire ne modifie pas le modèle. Il rend seulement l’interface existante plus publiable et plus lisible.

- **Probabilité de survie en groupes** : probabilité d’atteindre la phase à élimination directe depuis la phase de groupes.
- **Probabilité d’atteindre les huitièmes** : probabilité d’atteindre les huitièmes de finale.
- **Probabilité d’atteindre les quarts** : probabilité d’atteindre les quarts de finale.
- **Probabilité d’atteindre les demies** : probabilité d’atteindre les demi-finales.
- **Probabilité d’atteindre la finale** : probabilité d’atteindre la finale.
- **Probabilité de titre** : probabilité de remporter le tournoi.
- **Probabilité de finir vice-champion** : probabilité de perdre la finale.
- **Probabilité de finir troisième** : probabilité de gagner le match pour la troisième place.
- **Probabilité de finir quatrième** : probabilité de perdre le match pour la troisième place.
- **Pairwise** : surface complète des confrontations pour toutes les paires d’équipes uniques.
- **Fragilité** : dégradation des chances de tournoi sous scénarios adverses.
- **Upside** : hausse des chances de tournoi sous scénarios favorables.
- **Dépendance au parcours** : sensibilité des chances de tournoi à la route dans le tableau.
