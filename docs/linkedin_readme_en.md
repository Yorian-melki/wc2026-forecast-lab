# LinkedIn sharing note

This dashboard is designed to be presented as a quantitative side project.

Suggested positioning:
- reproducible local simulation project
- pairwise Monte Carlo layer
- full tournament propagation layer
- bilingual analytical interface
- model-vs-market disagreement view
- explicit limitations and no-certainty framing

What it is not:
- a betting automation tool
- a guarantee engine
- a replacement for official tournament data or final rosters
